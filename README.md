# CIFAKE Real vs AI-Generated Image Detection

This project detects real and AI-generated images using the Kaggle CIFAKE dataset.

## Dataset

Dataset: CIFAKE: Real and AI-Generated Synthetic Images  
Task: binary classification  
Image size: 32x32 RGB  

Labels:
- REAL = 0
- FAKE = 1
